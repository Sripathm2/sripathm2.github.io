# sripathm2.github.io

Personal academic and professional website for **Sripath Mishra** — CS PhD candidate at Purdue University working on machine learning for UAV network security and intrusion detection.

Live site: https://sripathm2.github.io

Built on [AcademicPages](https://github.com/academicpages/academicpages.github.io) (a Jekyll fork of Minimal Mistakes).

## Editing
- Site/author config: `_config.yml`
- About / home page: `_pages/about.md`
- Publications: `_publications/`
- Projects: `_portfolio/` (rendered at `/projects/`)
- CV: `_pages/cv.md`
- Navigation: `_data/navigation.yml`

## Local preview
```bash
bundle install
bundle exec jekyll serve -l -H localhost
```
